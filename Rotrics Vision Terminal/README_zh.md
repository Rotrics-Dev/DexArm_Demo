# 视觉终端

#### 介绍
视觉终端是为了更好更容易的使用前端视觉模块来做一些视觉相关的demo。目前支持分拣彩色木块和彩色豆子并将其放到指定位置。

#### 主要环境说明：

​	python3.6

​    opencv-python==3.4.1.15

​	pyserial>=3.1

​    PyQt5>=5.15

以上是主要的版本信息，可以使用 pip install package_name 进行安装。
